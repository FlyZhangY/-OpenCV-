时间：2019.12.15
用途：提取视频文件中指定的人物，找出该人物出现的时间，并将该人物的片段剪辑出来
用法说明：
	1，先在该目录下创建video文件夹，来存放需要提取的视频
	2，在该目录下创建imge文件夹，用来存放视频中提取的图片
	3，在该目录下创建face_imge文件夹，用来存放视频中的人脸图片
	4，在该目录下创建model文件用来存放训练模型
	5,   在该目录下创建face_train文件夹，用来存放训练集的图片，每一个人的图片存放在同一个文件夹下	
程序文件说明：

	1，最先执行Get_video_img.py文件，从视频中提取图片，每两帧提取一张图片，提取的图片存放在imge文件夹，图片以该帧出现的时间*100命名
	2，执行GET_face_img.py文件，将所有提取的图片中存在的人脸提取出来，命名方式该帧图片出现的时间*100_中人脸的序号命名，比如0010_1.jpg
	3,执行face_train.py文件，进行训练模型，将训练好的模型保存在model文件夹下，模型名为：ace_model_trained.xml。其中haarcascade_frontalface_default.xml文件是检测人脸的模型，在执行GET_face_img.py时使用
	4，执行face_predict.py文件，找出该人物出现的时间（s秒），并将该片段提取出来，存放在video路径下，可以自行更改。
